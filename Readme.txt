好吧这是一个游戏。。
游戏背景：
库尔斯克战役中的普罗诺夫卡，你扮演一名炮塔损坏不能旋转的虎式坦克车长（其实只是找不到炮塔旋转的素材），你需要对抗乌拉冲锋的三辆苏军T-34坦克和一辆租借而来的丘吉尔坦克。

操作方法：
点击黑线以下的方位可以左右移动你的虎式，键盘A和D键可以左右旋转你的炮塔，按空格键射击，射击之后需要2s进行装填，如果你的入射角度太小，那么敌人的坦克可能弹开你的炮弹。注意丘吉尔坦克的装甲较为厚重请尽量直击。
按1可以切换为AP弹，按2可以切换为HE弹（HE弹对坦克无效，对人员杀伤较好，但是我太懒不愿再把射击系统改成曲射的所以就没有安排苏军的步兵）

已知的Bug：
因为伪随机的关系exe文件运行的话每次的敌军布置位置是一样的，在Matlab中运行就没有这个问题。
